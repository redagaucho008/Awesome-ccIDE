## 🚀 Getting Started

When you open the application, you see a clean dashboard. If this is your first time using the tool, take a moment to look at the layout.

### Navigate the Dashboard

The main screen lists your recent sessions. Click on any session to see the terminal history and your saved notes. The search bar at the top allows you to find specific sessions by keywords. 

### Use the Sidebar

The sidebar sits on the right side of your screen. It contains five tabs:

1. **Skills**: A list of commands you use most often.
2. **Shortcuts**: Key combinations to move between windows.
3. **CLAUDE.md**: Tools to edit your central project rules.
4. **Launchd Tasks**: A place to see background tasks.
5. **Notes**: A text editor for your personal project thoughts.

### Organize Terminals

The quad-grid view lets you open many terminals at once. Select the grid icon in the top toolbar to split your screen. You can drag and drop terminals between grid spots. This setup works well for running multiple scripts while you read logs in another tab.

## 📋 Frequently Asked Questions

**Does this software store data online?**
Your session data stays on your computer. You control your files at all times.

**How do I clear my session history?**
Open the Settings menu, select Data, and click Clear History. This action removes all records of your past sessions from the application interface.

**Can I move the sidebar?**
The sidebar stays on the right side. This design keeps your terminal area wide and clear.

**What happens if the software freezes?**
Close the application and reopen it. The software saves your work every few seconds to prevent data loss.

**Does it require a constant internet connection?**
The interface works offline. You need an internet connection only if your Claude Code sessions require external calls.

## ⚖️ Troubleshooting

If you run into issues, check these common fixes.

### The Application Does Not Open
Sometimes, antivirus software blocks new applications. Check your security settings to ensure that Awesome-ccIDE has permission to run.

### Terminal Output Looks Wrong
If your text appears garbled in the terminal, check your system font settings. Using a standard monospace font like Consolas usually resolves display problems.

### Missing Session Data
Ensure you installed the application in a folder where you have write permissions. If you moved the application folder, the settings file might remain in the old location. 

### Performance Issues
If the app feels slow, close other programs that use many system resources. Using the quad-grid view with many active processes can use significant memory.
